## ⚡ 5-Minute Setup

### 1. Firebase Setup (2 minutes)
- [ ] Go to https://console.firebase.google.com
- [ ] Create new project: "fpop-inventory"
- [ ] Copy Firebase Config from Project Settings
- [ ] Paste into `js/firebase-config.js`

### 2. Enable Features (1 minute)
- [ ] Enable Authentication → Email/Password
- [ ] Enable Firestore Database → Production Mode

### 3. Create Admin (1 minute)
- [ ] Firebase Console → Authentication → Add User
- [ ] Email: admin@company.com (example)
- [ ] Password: (your choice)

### 4. Add Collections (1 minute)
Via Firestore Console, create:
- [ ] Collection: `users`
- [ ] Collection: `medications`
- [ ] Collection: `sales`

## 🐛 Troubleshooting

**Login doesn't work**
- Check Firebase config is correct
- Verify user exists in Firebase Auth
- Clear browser cache

**Data not saving**
- Check Firestore rules
- Verify collections exist
- Check browser console for errors

**Styles look wrong**
- Check css/style.css is linked
- Verify Bootstrap CDN loads
- Clear browser cache

**Can't upload Excel**
- Check file format (.xlsx, .xls, .csv)
- Verify column headers match
- Check browser console for errors
