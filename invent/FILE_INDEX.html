🚀 Getting Started

Before launching the system, follow these steps:

  1. Read the SETUP_GUIDE.md - Contains detailed setup instructions
  2. Update Firebase Config - Edit js/firebase-config.js with your Firebase credentials
  3. Create First Admin User - Use Firebase Console to create initial admin
  4. Open index.html - Start using the system!

🗄️ Firestore Collections

Required Collections

Create these collections in your Firestore database:

  • /users - Stores user information (uid as document ID)
  • /medications - Stores medication inventory
  • /sales - Stores sales transactions

See README.md for detailed Firestore schema documentation
