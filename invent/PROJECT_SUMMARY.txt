## 🚀 GETTING STARTED

### Step 1: Update Firebase Configuration
```
File: js/firebase-config.js
Replace the firebaseConfig object with your Firebase credentials
```

### Step 2: Set Up Firebase Project
1. Go to https://console.firebase.google.com
2. Create new project or use existing
3. Enable Authentication (Email/Password)
4. Enable Firestore Database
5. Create collections: users, medications, sales

### Step 3: Create First Admin User
1. Use Firebase Console to create admin user
2. Add user document to Firestore /users collection
3. Set role: "admin"

### Step 4: Launch Application
- Open index.html in browser
- Start with login
- Begin managing inventory!
